# Русификация DawnCraft — Echoes of Legends

Эта папка — **единый оверлей со всеми изменёнными файлами**. Её можно скопировать
на другую копию модпака и быстро применить русификацию.

## Что внутри
- `resourcepacks/RU_DawnCraft/` — ресурспак с русскими `ru_ru.json` для модов,
  у которых не было русского языка (~109 модов).
- `config/ftbquests/` — переведённые квесты FTB Quests.
- `apply.ps1` — копирует всё это в `minecraft/` текущего инстанса.
- `GLOSSARY.md` — единая терминология.
- `PROGRESS.md` — что уже переведено, что осталось.

## Как применить
**Проще всего:** двойной клик по **`Применить_русификацию.bat`** (спросит подтверждение и применит).

Или вручную:
```powershell
powershell -ExecutionPolicy Bypass -File apply.ps1
```
Затем в игре: **Настройки → Наборы ресурсов** → включить **DawnCraft RU**
(поставить выше `DawnCraft_Resources`). Перезапустить игру.

## Принцип
- Моды переводятся через ресурспак (не трогаем jar'ы) — чисто и обновляемо.
- Квесты переводятся прямо в SNBT (другого механизма в этой версии FTB Quests нет).
- 88 модов уже имеют русский внутри jar — их не трогаем.
- 101 мод не имеет переводимых строк (библиотеки) — пропущены.
