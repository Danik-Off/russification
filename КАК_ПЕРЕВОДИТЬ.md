# Как переводить DawnCraft и где продолжить

Это пошаговая инструкция по русификации модпака **DawnCraft — Echoes of Legends**
(Minecraft 1.18.2, Forge). Всё, что нужно, лежит в папке `russification/`.

---

## 0. Краткая карта папки

```
russification/
├─ resourcepacks/RU_DawnCraft/        ← ПЕРЕВОД МОДОВ (ресурспак)
│   ├─ pack.mcmeta
│   └─ assets/<mod_id>/lang/ru_ru.json   ← редактируешь ЭТИ файлы
├─ config/                            ← ПЕРЕВОДЫ ЧЕРЕЗ КОНФИГ
│   ├─ ftbquests/quests/chapters/*.snbt        ← квесты (SNBT)
│   ├─ dcclasses/assets/dawncraft/lang/ru_ru.json  ← классы DawnCraft (готово)
│   └─ fancymenu/locals/ru_ru.local            ← кастомные меню (готово)
├─ metadata/                          ← изменения вне minecraft/ (instance.cfg, manifest)
├─ _СНИМОК_ПЕРЕВЕДЕНО/                ← копия только готовых файлов (для переноса)
├─ apply.ps1                          ← применить всё к игре одной командой
├─ PROGRESS.md                        ← ЧЕК-ЛИСТ: что готово, что осталось
├─ GLOSSARY.md                        ← единые термины (свериться перед переводом!)
├─ КАК_ПЕРЕВОДИТЬ.md                  ← этот файл
└─ .scan/                             ← результаты сканирования модов (служебное)
```

**Источник истины — папка `russification/`.** Игра НЕ читает её напрямую:
файлы попадают в игру только после запуска `apply.ps1`.

---

## 0.1 Что уже было сделано ДО (учтено и встроено)

Часть локализации существовала в `flame/overrides/` ещё до текущей работы — она
перенесена в `russification/` и теперь разворачивается через `apply.ps1`:

- **Классы DawnCraft** → `config/dcclasses/assets/dawncraft/lang/ru_ru.json`
  (Берсерк, Воин, Копьеносец, Ронин, Танк, Астролог, Уличный боец, Странник + описания).
  ⚠️ В исходном файле ключи были НЕВЕРНЫЕ (`classes.dawncraft.*` и `dc_classes.*`),
  из-за чего перевод не применялся. Исправлено на реальные ключи **`class.dawncraft.*`**
  (см. `config/dcclasses/assets/dawncraft/lang/en_us.json`). Урок: всегда сверяй ключи с en_us.
- **Кастомные меню** (главное меню, экран загрузки, мультиплеер, FTB Quests, DC Classes)
  → `config/fancymenu/locals/ru_ru.local`. Это механизм FancyMenu — **НЕ** json в ресурспаке.
  Поэтому мод `fancymenu` (1229 строк редактора) из перевода ИСКЛЮЧЁН — заготовка удалена.
- **Метаданные** (имя инстанса, поля `*_ru` в manifest) → описаны в `metadata/МЕТАДАННЫЕ.md`.

DC-Classes — пример мода, который грузит язык не из jar, а из `config/.../assets/<ns>/lang/`.
Если встретишь похожий — клади перевод в `config/`, а не в ресурспак.

## 1. Два разных способа перевода

### A. МОДЫ → через ресурспак (jar'ы не трогаем!)
Файлы: `resourcepacks/RU_DawnCraft/assets/<mod_id>/lang/ru_ru.json`

Это обычный JSON «ключ: значение». **Переводим только значение справа**, ключ слева не трогаем:
```jsonc
{
  "item.simple_mobs.giant_knot": "Узел великана",   // ← перевели значение
  "entity.simple_mobs.fire_giant": "Огненный великан"
}
```
Сейчас в этих файлах лежит английский текст (заготовки). Заменяешь английские
значения на русские прямо в файле.

### B. КВЕСТЫ → правка SNBT на месте
Файлы: `config/ftbquests/quests/chapters/*.snbt`

Переводим только текстовые поля:
- `title: "..."`      — название
- `subtitle: ["..."]` — подзаголовок
- `description: [ "...", "...", "..." ]` — описание (массив строк, каждая строка = абзац)

Пример:
```
title: "Getting Started"      →   title: "С чего начать"
```

---

## 2. ЧТО НЕЛЬЗЯ ТРОГАТЬ (важно — иначе сломается)

- **Ключи** в JSON (слева от `:`).
- **`id:`**, `filename:`, `type:`, `icon:`, `item:`, любые коды вида `"4873DF41F2864A04"`.
- **Плейсхолдеры:** `%s`, `%d`, `%1$s`, `{0}` — оставляем как есть, порядок не меняем.
- **Переносы строк** `\n` и `\\n` — оставляем как есть.
- **Цветовые коды** `§a`, `§6`, `&c` и escape `\\&` — не трогаем.
- **Теги** в угловых скобках `<...>` и ID предметов `minecraft:cooked_beef`.
- **Команды и клавиши:** `/spawn`, `F3` — не переводим.

Пример с плейсхолдерами:
```
"You need %s coins"   →   "Вам нужно %s монет"
```

---

## 3. Терминология

Перед переводом свериться с **`GLOSSARY.md`**. Главное:
Boss→Босс, Stamina→Выносливость, Spell→Заклинание, Reforge→Перековка,
Socket→Гнездо, Villager→Житель, Dungeon→Подземелье, Stronghold→Крепость.
Обращение к игроку — на «вы», тон нейтральный.
Новые устойчивые термины и имена боссов добавляй в `GLOSSARY.md`, чтобы перевод не «плавал».

---

## 4. Кодировка (критично!)

- `.json` и `.snbt` сохраняй в **UTF-8** (без BOM — норм). Иначе кириллица превратится в «кракозябры».
- `.ps1` (если правишь) — только ASCII в коде, иначе Windows PowerShell 5.1 не распарсит.

---

## 4.1 Целостность SNBT и восстановление

Квест-файлы `.snbt` ломаются, если нарушить баланс скобок `{ }` и `[ ]` (например,
случайной вставкой/дублированием в редакторе). Битый файл рушит ВСЮ книгу квестов.

- Проверка баланса одного файла:
  ```bash
  F=config/ftbquests/quests/chapters/dawncraft.snbt
  echo "{ $(tr -cd '{' < "$F" | wc -c) } $(tr -cd '}' < "$F" | wc -c)"   # числа должны совпасть
  ```
- `apply.ps1` сам проверяет ВСЕ `.snbt` перед копированием и НЕ развернёт битые
  (выведет список и прервётся).
- **Чистые оригиналы** всех квестов лежат в `minecraft/defaultconfigs/ftbquests/quests/`.
  Если файл повредился — восстанови из defaultconfigs и переведи заново:
  ```bash
  cp minecraft/defaultconfigs/ftbquests/quests/chapters/ИМЯ.snbt \
     russification/config/ftbquests/quests/chapters/ИМЯ.snbt
  ```
- НЕ редактируй большие `.snbt` вручную в IDE без нужды — правь только текст в кавычках
  у полей `title`/`subtitle`/`description`, ничего не дублируя.

## 5. Как применить к игре

**Двойной клик по `Применить_русификацию.bat`** — спросит подтверждение и применит.
Либо вручную:
```powershell
powershell -ExecutionPolicy Bypass -File apply.ps1
```
Затем в игре: **Настройки → Наборы ресурсов** → включить **DawnCraft RU**,
поставить его **выше** `DawnCraft_Resources`. Перезапустить игру.
Квесты подхватываются автоматически после перезапуска.

## 6. ГДЕ ПРОДОЛЖИТЬ (порядок работ)

Сверяйся с **`PROGRESS.md`** — там живой чек-лист. Текущее состояние:

### Готово
- Вся инфраструктура, заготовки `ru_ru.json` всех 109 модов.
- Названия всех вкладок квестов, групп глав, таблиц наград.
- Главы `a_little_help` и `monster_coins` — полностью.
- Классы DawnCraft (`config/dcclasses`) — исправлены ключи, перевод рабочий.
- UI-строки экрана классов (`assets/dcclasses` в ресурспаке) — переведены.
- Меню FancyMenu (`config/fancymenu/locals/ru_ru.local`) — готово (из прошлой работы).

### Следующий шаг — КВЕСТЫ (приоритет №1, это видит игрок)
Доперевести описания внутри глав, по одной. Рекомендуемый порядок:
1. `dawncraft.snbt` (вкладка «С чего начать») — первое, что читает новичок.
2. `bossesminibosses_in_this_realm.snbt` — боссы.
3. `exploration.snbt`, `magic.snbt`, `enchanting.snbt`,
   `reforging_and_socketing.snbt`, `storage.snbt`, `witchcraft.snbt`,
   `lost__found_quest_map_replacements.snbt`, `useful_commands_for_servers.snbt`.
4. `test.snbt` — служебная, скрытая, НЕ переводим.

Как найти непереведённые строки в главе (английский текст в полях title/subtitle/description):
```bash
grep -nE '^\s*(title|subtitle): "|^\s*"' config/ftbquests/quests/chapters/dawncraft.snbt
```

### Потом — МОДЫ (приоритет №2)
Идём по списку в `PROGRESS.md`, **сверху вниз** (отсортировано по объёму строк).
Геймплейные моды (важные, делать первыми):
`simple_mobs`, `dannys_expansion`, `enlightened_end`, `jet_and_elias_armors`,
`dawncraft`, `nourished_nether`, `goblinsanddungeons`, `lilwings`, `meetyourfight`,
`scattered_weapons`, `undead_revamp2`, `restored_earth`, `nethersdelight`.

В САМОМ конце (низкий приоритет, не геймплей):
`drippyloadingscreen`, `bhmenu`, библиотеки (`libx`, `creativecore`,
`structure_gel`, `geckolib3` и т.п.).
**`fancymenu` НЕ переводим** через ресурспак — меню уже сделаны через `.local` (см. п.0.1).

Найти следующий непереведённый мод (файл ещё полностью на английском):
```bash
cd resourcepacks/RU_DawnCraft/assets
for d in */lang/ru_ru.json; do
  grep -qP '[А-Яа-я]' "$d" || echo "НЕ НАЧАТ: $d"
done
```

### После каждого куска
1. Отметь галочку в `PROGRESS.md` (`[x]` готово, `[~]` частично).
2. Скопируй готовый файл в `_СНИМОК_ПЕРЕВЕДЕНО/` (по тому же пути) — для переноса.
3. Запусти `apply.ps1` и проверь в игре.

---

## 7. Перенос на другую копию модпака

Скопировать всю папку `russification/` в корень нужного инстанса
(рядом с `minecraft/`) и запустить `apply.ps1`. Готово.
Либо взять только `_СНИМОК_ПЕРЕВЕДЕНО/` + ресурспак, если нужно лишь готовое.
